Sapta Samudra

Screenshot v0.2.0 (click to see more screenshots):


Sapta Samudra is a game adapted from PopCap Seven Seas, built as the project assignment for CS2103, Software Engineering.
  • Download latest stable Sapta Samudra (v0.2.2)
  • Map for puzzles
  • User guide on how to play Sapta Samudra
  • Map Editor (for advanced users)
You can download the Map for Puzzles (right click, "Save link as") and save it on the same folder, then play with the name "Puzzle 1" (without the quotes) to play prototype of puzzle mode. You will have only one life and no weapon. You will need to kill all the enemies, as usual. Can you win all 5 puzzles?

You can also download and play previous versions. Please note that these version may not be compatible with your current data, so it's better to play them on separate folder.
