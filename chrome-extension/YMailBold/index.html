Yahoo! Mail Folder Bolder - Chrome Extension

16 Dec 2012



 The new Yahoo! Mail UI does not make the folders on the left panel bold whenever there are unread e-mails in them. This can make the e-mails go unnoticed, therefore I make this really simple Chrome Extension to make it in bold.
